# Nami Bot Listing Information

## Top.gg Listing

### Short Description (102/120 characters)
AI chat, expressive TTS, web search, games, and messaging automation for Discord communities.

### Long Description
Nami brings AI-powered chat, expressive voice synthesis, web search, mini-games, and server automation to your Discord community—all in one friendly bot.

**Chat & AI**
• Mention @Nami to chat naturally in any channel
• Choose between Smart mode (OpenRouter) for polished responses or Uncensored mode (Ollama) for open outputs
• Web search integration to find and summarize real-time information
• Conversation memory keeps context across sessions

**Messaging Workflows**
• Send messages across channels with `@Nami send msg to #channel say ...`
• Keep text exactly as written with "don't edit" modifier
• Optional AI polish for clarity and flow

**Voice & TTS**
• Expressive Google Text-to-Speech with tone-aware delivery
• Smart text cleanup for slang, emojis, and natural pronunciation
• Voice channel join/leave, queue controls, auto-read, and configurable auto-join
• Per-user voice, speed, and language preferences

**Customization & Storage**
• Personal settings: language, AI model, voice ID, search preference
• Optional Supabase-backed storage for settings, history, and TTS cache
• Server-level feature flags, system prompts, and announcements
• Daily usage limits with fair-use tracking

**Built for Production**
• Health checks and fallback handling
• Works with local Ollama or Ollama Cloud for uncensored mode
• Reliable enough for community-scale deployment

Use `/help` in Discord to explore all commands. Perfect for communities that want intelligent automation without complexity.

### Prefix
`@Nami` and `/`

Nami is mention-based for natural chat (`@Nami hello`) and uses slash commands for specific features (`/ask`, `/search`, `/voice`, `/tts`, `/preferences`, `/admin`, `/game`, `/memory`).

Primary prefix: `@Nami`

What the bot actually does

A full-featured Discord bot built for utility, speed, and reliability.

01

AI that can chat, search, remember, and switch personality.

Mention Nami for a conversational reply, ask direct questions with /ask, pull live search summaries with citations, and switch between Smart and Uncensored model modes.

  • Mention chat plus slash-command Q&A
  • DuckDuckGo-backed search summaries
  • Per-user style, language, and search defaults
  • Conversation memory with view and clear controls
02

Voice that sounds more human, not just louder.

Nami uses Google Cloud TTS with expressive handling for punctuation, slang, emojis, queueing, playback control, and voice-channel automation.

  • /tts say, skip, stop, clear queue, and inspect status
  • Saved voice IDs, playback speed, and language preferences
  • Auto-read and auto-join options for voice channels
  • Daily usage controls for users, guilds, and global limits
03

Automation and admin tools that fit real communities.

Cross-channel messaging, announcements, feature flags, prompts, and health endpoints make Nami easy to run for creators, friend groups, and larger servers.

  • @Nami send msg to #channel say ... workflows
  • Server-level admin toggles and announcement routing
  • Local JSON or Supabase-backed storage
  • Render-ready health checks and keepalive support
